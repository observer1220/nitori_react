### 顧客前台 (Customer Portal)

- 登入
  [x] 顧客登入頁：使用者登入時，傳輸明碼在後端比對，正確時放行，失敗則顯示密碼錯誤
  [] 忘記密碼
- 註冊
  [] 顧客註冊
  [] 驗證碼
- 首頁
  [] 展示熱門產品、促銷商品
- 產品列表頁
  [] 瀏覽及篩選產品（依分類、價格區間等）
- 產品詳情頁
  [] 查看產品圖片、描述、價格、庫存狀態
  [] 加入購物車
- 購物車頁面
  [] 顯示已加入購物車的產品，更新數量或刪除項目
- 結帳頁面
  [] 提供顧客輸入配送地址、付款方式，並確認訂單
- 訂單歷史頁
  [] 顯示顧客的歷史訂單及訂單詳情
